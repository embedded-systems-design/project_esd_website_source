Exporting Gerber files from Cadence PCB Editor

By Dan Aukes 01 January 0001

What is a Gerber file?

A Gerber file (also known as artwork) is a 2-D graphical representation of a single layer of a PCB. A typical design will have individual Gerber files for each layer (e.g., top copper, bottom copper, top silkscreen, bottom silkscreen, top soldermask, bottom soldermask) of a PCB.

What is a drill file?

A drill file (also known as a NC Drill file) stores both the specific sizes and types of drill bits that will be used in manufacturing a PCB in addition to the specific coordinate locations where each hole must be drilled. Our equipment uses files that are in Excellon format.

You need both Gerber files for each layer and a single drill file in order to successfully submit your design for manufacturing.

How do I export Gerber files from Cadence?

  1. Open your PCB layout in Allegro PCB Designer

  2. Choose “Manufacture > Artwork…” The Artwork Control Form window (see Figure 1) appears.

{class=“img-fluid”} Figure 1: Artwork window

  1. Next, a board outline must be added. Right-click on the TOP folder and choose “Add Manual” (see Figure 2).

{class=“img-fluid”} Figure 2: Right-click on TOP and choose “Add Manual”

  1. Enter a film name of OUTLINE and click “OK” (see Figure 3).

{class=“img-fluid”} Figure 3: Film Name window

4.5. Expand OUTLINE folder and right click on “BOARD GEOMETRY/DESIGN_OUTLINE” and select add.

{class=“img-fluid”} Figure 3.5: Subclass Selection window

  1. In the Subclass Selection window (see Figure 4), expand the BOARD GEOMETRY folder and check the box next to DESIGN_OUTLINE. Click “OK”.

{class=“img-fluid”} Figure 4: Subclass Selection window

  1. Select the OUTLINE checkbox (see Figure 5). Make sure in Film options that “Film name: DESIGN_OUTLINE” appears (if it does not, select the OUTLINE checkbox again). Set the Undefined line width to 0.1.

{class=“img-fluid”} Figure 5: Artwork Control Form

  1. If you have an anti-etch (rubout), you will need to add it to the layer that you put it on. Click on the down arrow under the layer folder(s) that you added it too. Then, right-click on the items in the folder and select Add (see Figure 6).

{class=“img-fluid”}

Figure 6: Add anti-etch layer

  1. Then, expand the anti-etch subclass folder and select the box next to the layer you are adding it to (see Figure 7). Repeat this process for each layer that you have an anti-etch on.

{class=“img-fluid”}

Figure 7: Adding the anti-etch layer to the top

  1. Click the “Select all” button to output all of the layers

  2. Click “Create Artwork” to export the Gerber files

  3. Click “OK” to return to the layout

How do I export drill files from Cadence?

  1. Open your PCB layout in Allegro PCB Designer

  2. Choose “Manufacture > NC > NC Drill…” The NC Drill window (see Figure 6) appears.

    {class=“img-fluid”}

    Figure 6: NC Drill window

  3. Click “Parameters…” The Parameters window (see Figure 7) appears.

    {class=“img-fluid”} Figure 7: NC Parameters window

  4. Select “Enhanced Excellon format” and click “Close”

  5. Name the drill file and save it in the same directory as your project

  6. Select “Auto tool select” and “Repeat codes”

  7. Click “Drill” to export the drill file

  8. Click “Close” to return to the layout
